Overview

bibliothiki (βιβλιοθήκη, ancient Greek for “library”) is a set of common re-usable capabilities that can be shared between different projects.

All APIs and packages provided by bibliothiki are:

  • broadly applicable (i.e., not overly specific),
  • adaptable,
  • modular,
  • well-tested,
  • dependency free (or minimal dependencies) (jspecify is always a minimum).
